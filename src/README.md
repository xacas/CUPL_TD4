# src
CUPLのTD4ソースコードファイルです。
ピンの宣言部以外はあえて空欄にしてあるので、
ぜひご自身で実装してみてください。

テストベンチは以下の2つがあります。

* WinCUPLのシミュレーションテストファイル(.si)
* Arduinoを使った実機テスト

前者は各ディレクトリに配置しています。
後者は test ディレクトリをご覧ください。

シミュレーションはWinCUPLⅡで動作確認していますが、
もしWinCUPL （Ⅱじゃない方）の Winsim で実行すると、以下のような波形も出せます。
